# Fairy Catch

An arcade game where you catch falling apples. Built with Godot 4.

![gameplay screenshot](fcatch.png)

## Play

Grab the latest build from [itch.io](https://terra2o.itch.io/fairy-catch).

## Build from source

1. Clone the repo
2. Open `project.godot` in Godot 4
3. Hit run or export
